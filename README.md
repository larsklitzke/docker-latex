# Dockered LaTeX

Dockered LaTeX version based on Ubuntu with Biber preinstalled.

```
docker pull larsklitzke/latex
```

The following packages are installed

* texlive 
* texlive-latex-recommended
* texlive-latex-extra
* texlive-bibtex-extra
* texlive-fonts-extra
* texlive-lang-german
* texlive-scientific
* texlive-math-extra
* biber
